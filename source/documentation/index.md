# CTC Traders API roadmap

Last updated: November 2025

### What we’re working on now

**NCTS Phase 6**
- We have completed the DDNTA analysis and are now progressing with initial API build activities for Phase 6, along with updates to key technical documents, including the TIS, Service Guide, and XSDs,  to ensure SWD integration readiness.

### Upcoming Changes
**March 2025 (Q1): Decommissioning of CTC API Version 2.0**
- Version 2.0 of the CTC API will no longer be usable after January 21, 2025. Official deprecation activities will begin in March 2025.
